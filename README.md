### **Maiden Alert**
## ✨ Settings _(`/malert`, `/maidenalert`)_

  - Disable Sound: Disable/Enable sound notification
  - Message Alert: Disable/Enable chat message notification
  - Sound: Choose a sound from the list to be played as the notification sound if enabled
  - Test Notification: Press to simulate the notification happening _(Following settings)_

## 🧾 Notes

- Everytime that you are doing a **Fate** and any type of **Forlon Maiden** spawns, you will receive a notification accordinly to your `/malert` settings.

Don't miss any Maiden spawn anymore!
